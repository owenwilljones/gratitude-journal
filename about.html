About Gratitude Journalling and this website

What is Gratitude Journalling?

Gratitude journalling is a wellbeing exercise that involves listing things you are grateful of. Things you are grateful for, or "gratitudes", can be anything from having a steady income to the taste of a cup of coffee in the morning.

You can learn more about what it is and why it's useful by watching this video.

What is this website?

This website is intended as a simple to use platform to enable you to practice gratitude journalling. You can use the homepage to enter any number of gratitudes and save said gratitudes for up to a week.

This website is open source. You can check out the code or flag issues with this site by going here.
